# Power Follow-Ups: Refining AI Outputs

The first AI response is rarely the final one. Power Follow-Ups are a set of moves that let you shape and improve any AI output quickly. They work because they're specific — each one targets a different dimension of quality.

## The Nine Power Follow-Ups

**Simplify** — "Make this simpler." Use when the output is technically correct but too complex for your audience or use case. Works especially well for converting expert language into plain language, or shortening a response that's too long.

**Sticky** — "Give me an analogy that makes this stick." Use when you need to explain a concept to someone unfamiliar with it. Analogies anchor abstract ideas to something the listener already understands.

**Test** — "What if we changed [variable]?" Use when you want to explore how the output would change under different conditions. "What if the audience was frontline staff instead of supervisors?" "What if we had no budget for this?" Forces the AI to think through contingencies.

**Push** — "Find what's missing." Use when you suspect the output is incomplete or has gaps. The AI will look for overlooked considerations, unstated assumptions, or aspects of the problem you didn't address.

**Flip** — "Make the strongest case against this." Use when you want to pressure-test an argument or plan. The best way to find weaknesses is to actively argue for the other side. This is the Critique move in follow-up form.

**Rank** — "Rank these by impact." Use when you have multiple options and need to prioritize. Ask the AI to evaluate the list on whatever dimension matters most — feasibility, impact, urgency, cost, risk.

**Ground** — "Give me a specific example I could use Monday." Use when the output is too abstract. Forces the AI to get concrete — a real scenario, a specific sentence, an actionable step rather than a general principle.

**Tone** — "Rewrite this for a different audience." Use when the content is right but the register isn't. "Rewrite this for the county board." "Write this like a letter to families, not a policy memo."

**Format** — "Turn this into a [table / checklist / one-pager / bullet list]." Use when the default format doesn't match your needs. Format changes can radically improve how a good response lands.

## How to Use Power Follow-Ups

Pick the 1-2 that fit the moment. Don't list all nine — overwhelming options is worse than none.

The best time to offer a follow-up is immediately after a response, when the user is looking at the output and thinking about what to do with it. Anticipate what they'd want to improve or adapt.

Power Follow-Ups can chain. You might simplify an output, then reformat it, then ask for a specific example. Each iteration brings the output closer to what the user actually needs.
